## Example Report Structure

All example reports follow this structure:

1. **Header**: Brand, period, analysis date, report ID
2. **Executive Summary**: Overall grade, key findings, immediate actions
3. **Context & Scope**: Analysis parameters, channels, frameworks
4. **Scores**: Dimension-by-dimension breakdown with evidence
5. **Findings & Risks**: Detailed analysis, strongest/weakest areas
6. **Improvement Roadmap**: Prioritized recommendations (effort × impact)
7. **Limitations & Certainty**: Data and analysis constraints
8. **Sources**: Full citation list (academic + industry)
